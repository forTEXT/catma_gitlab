GitMA Metrics
=============

.. note::

    These are integrated in the ``CatmaProject`` class.

Examples to use the metrics functions can be found in the `demo notebook <https://github.com/forTEXT/gitma/blob/main/demo_notebooks/inter_annotator_agreement.ipynb>`_.
